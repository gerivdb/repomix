# VERSES-HUB

Dépôt central de l'Architecture Diamant pour la gestion des verses.

## Structure

- `src/` - Code source des outils core
- `docs/` - Documentation
- `registry/` - Registry JSON-LD des verses
- `policies/` - Politiques de gouvernance

## Installation

```bash
pip install -r requirements.txt
```

## CI/CD

GitHub Actions configuré dans `.github/workflows/`
